# Yen Lee - Personal Portfolio

A sleek, modern, and high-contrast personal web page inspired by the premium 'Adelina' template. This project features a dark aesthetic, geometric typography, and a real-time clock.

## Features

- **Adelina-Inspired Design**: A professional, minimalist dark theme with sleek purple accents and abstract geometric patterns.
- **Dynamic Clock**: a real-time digital clock integrated into the navigation bar.
- **Micro-Animations**: 
  - Smooth text-reveal animations on page load.
  - Interactive mouse-parallax effect on the background.
  - Dynamic hover effects on key elements.
- **Fully Responsive**: Optimized for seamless viewing across desktops, tablets, and mobile devices.

## Technology Stack

- **HTML5**: Semantic structure for better SEO and accessibility.
- **Vanilla CSS**: Custom styling with modern features like Backdrop Filter, Flexbox, and CSS Animations.
- **JavaScript**: Lightweight logic for the real-time clock and interactive parallax effect.
- **Assets**: High-quality abstract background generated to match the portfolio's aesthetic.

## Project Structure

- `index.html`: The main entry point of the website.
- `style.css`: Contains all design tokens, layouts, and animations.
- `script.js`: Handles time updates and interactive effects.
- `background.png`: Custom abstract background asset.

## How to View

Simply open `index.html` in any modern web browser to view the portfolio.
